Change in tariff heading (CTH)

When a non-originating material is used to produce another finished product, the 4-digit level of its original commodity code will need to change once itâ€™s incorporated into the final product. This is referred to as a change in tariff heading.

Example:

Thin copper wire with a diameter below 6mm comes under the commodity code 740819.

A UK manufacturer makes wire from copper sheets sourced from outside the UK.

The 6-digit commodity code for the copper sheets is 740919

The 6-digit commodity code for the thin UK wire is 740819

The commodity code changes at the 4-digit level when the copper sheets become wire. This means it has gone through a change in tariff heading.
